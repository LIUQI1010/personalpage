# 个人简历网站

一个用于新西兰求职的现代化个人网站。

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **UI组件库**: shadcn/ui
- **样式**: Tailwind CSS
- **动画**: GSAP
- **语言**: JavaScript
- **部署**: Vercel

## 开发环境设置

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 项目结构

```
src/
├── components/     # React组件
├── hooks/          # 自定义Hook
├── lib/            # 工具函数
├── utils/          # 通用工具
├── App.jsx         # 主应用组件
└── main.jsx        # 应用入口
```

## 部署

项目配置为自动部署到Vercel，推送到GitHub仓库即可自动部署。

## 功能特性

- 响应式设计
- 现代化UI组件
- 流畅的动画效果
- 移动端友好
